TORI.

O Tori é uma plataforma que gerencia as principais ferramentas de engenharia de software. O intuito da plataforma é ser um ambiente que possa reunir as principais ferramentas que poderão auxiliar qualquer desenvolvedor no processo de criação dos modelos de processos de software, na análise, especificação, validação e gestão de requisitos, ferramentas para testes de software, prototipação, gerenciamento de projetos, ferramentas base para framework ágil e quaisqueres outras ferramentas que sejam úteis para o desenvolvimento de qualquer aplicação.

Asana

O Asana é uma plataforma de gestão de trabalho e colaboração baseada na web, projetada para ajudar equipes a organizar, acompanhar e gerenciar projetos e tarefas de forma eficiente. Com uma interface intuitiva e recursos poderosos, o Asana permite que os usuários criem projetos, atribuam tarefas, definam prazos, compartilhem arquivos e comuniquem-se facilmente, tudo em um único lugar.

Jira

O Jira é uma plataforma popular de gerenciamento de projetos e acompanhamento de problemas, desenvolvida pela Atlassian. Projetado para equipes ágeis de desenvolvimento de software, o Jira permite que os usuários criem, priorizem e atribuam tarefas, acompanhem o progresso do trabalho e colaborem de forma eficaz em um ambiente centralizado.

Trello

O Trello é uma plataforma de gerenciamento de projetos baseada em quadros, criada pela Atlassian. Usando um sistema visual de cartões e listas, o Trello permite que equipes organizem e priorizem tarefas, acompanhem o progresso do trabalho e colaborem de forma eficaz. Os usuários podem criar cartões para representar tarefas ou itens a serem feitos e movê-los entre listas para indicar seu status atual, como "A Fazer", "Em Andamento" e "Concluído".

Eclipse

O Eclipse oferece uma ampla gama de recursos para desenvolvedores, como edição de código com realce de sintaxe, depuração, compilação, testes e integração com sistemas de controle de versão, como Git e Subversion.

NetBeans

O NetBeans oferece uma variedade de recursos avançados, como edição de código com realce de sintaxe, refatoração, depuração, análise de código estática, suporte a testes e integração com sistemas de controle de versão.

Visual Studio Code

O VS Code oferece recursos poderosos, como realce de sintaxe, conclusão de código inteligente, depuração integrada, controle de versão com Git integrado, além de uma série de ferramentas para desenvolvimento web, como um navegador embutido e suporte a extensões para frameworks populares como React e Vue.js.

Bitbucket

Bitbucket é uma plataforma de hospedagem de código baseada em nuvem, fornecida pela Atlassian. Ela oferece controle de versão baseado em Git ou Mercurial para projetos de software, permitindo que equipes colaborem no desenvolvimento de código-fonte de forma eficiente.

Git

O Git é um sistema de controle de versão distribuído amplamente utilizado para o gerenciamento de código-fonte em projetos de software. Desenvolvido por Linus Torvalds, o criador do Linux, o Git é conhecido por sua velocidade, eficiência e capacidade de lidar com projetos de qualquer tamanho com facilidade.

GitHub

O GitHub é uma plataforma de hospedagem de código-fonte baseada na web, que utiliza o sistema de controle de versão Git. É amplamente utilizada por desenvolvedores para hospedar projetos de software, colaborar com outros desenvolvedores, revisar código, gerenciar problemas e realizar integração contínua/desenvolvimento contínuo (CI/CD).

>

Confluence

Confluence é uma plataforma de colaboração baseada na web, desenvolvida pela Atlassian, que permite às equipes criar, compartilhar e colaborar em documentos, páginas wiki, documentação de projetos, especificações de produtos e muito mais.

IBM (DOORS Next)

O IBM Engineering Requirements Management DOORS Next é uma ferramenta avançada de gestão de requisitos desenvolvida pela IBM. Ele oferece uma abordagem moderna e baseada na web para captura, análise, rastreamento e gerenciamento de requisitos em todo o ciclo de vida do desenvolvimento de produtos ou sistemas.

RequisitePro

O RequisitePro é uma ferramenta de gestão de requisitos desenvolvida pela IBM. Projetado para auxiliar equipes de desenvolvimento de software e engenheiros de sistemas, o RequisitePro simplifica o processo de captura, análise, rastreamento e gerenciamento de requisitos ao longo do ciclo de vida do projeto.
