Nome do Projeto

Bem vindo a Nome do Proejeto, um site que apresenta as principais ferramentas de engenharia de software.

Ferramentas De Controle de Versão

GitHub

Github é uma plataforma feita em nuvem onde você pode armazenar, compartilhar e trabalhar com outros programadores.

Git

Git é um aplicativo de controle de versão, projetado para aguentar projetos de pequena ou larga escala com rápidez e eficiência.

BitBucket

O Bitbucket Cloud é uma ferramenta de hospedagem de código e colaboração baseada em Git, criada para equipes.
